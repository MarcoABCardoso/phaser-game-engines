# Starter onboarding sessions

Release 1 requires five observed sessions with developers unfamiliar with the
implementation. This file is the evidence log; do not substitute maintainer
runs or automated verification for user evidence.

## Protocol

Ask each participant to start from a clean environment and the public registry.
Do not coach unless they abandon a step. Time these milestones:

1. Run the generator and start the game.
2. Identify the objective and complete/restart the loop.
3. Add one piece of content and one game-owned rule.
4. Run `npm run verify` and produce a deployable build.

Record errors verbatim, abandoned steps, documentation consulted, any toolkit
source copied or edited, device/input choice, language, Node/browser versions,
and whether the facilitator intervened.
