Choose the shortest path

Start a new game or add the toolkit to an existing Phaser project.

Generate a starter

Create a small runnable project with a genre, language, and input adapter selected up front.

npm create @phaser-game-engines/game -- my-game --genre platformer

Use the toolkit

Import only the core, genre, recipe, headless, or content subpaths your game needs.

npm install @phaser-game-engines/toolkit phaser

Support status

Public 0.x prerelease. The toolkit is installable and tested across Node 20.19, 22, and 24. APIs may still change with documented migration notes before 1.0.
