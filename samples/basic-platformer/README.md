# Basic platformer movement sandbox

This sample demonstrates `PlatformerScene` traversal and recipe tuning. It is
intentionally not a complete game: it has no objective, stage-completion rule,
result scene, saving, or restart loop.

Use the recommended `@phaser-game-engines/create-game` platformer template when
you want a complete title → play → result starter with explicit content, rule,
scene, entity-presentation, and input ownership.
